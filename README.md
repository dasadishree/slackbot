# HOW TO USE
- TO START (RUNS IN THE BACKGROUND): cd to the right folder, nohup python3 tracker.py > /dev/null 2>&1 &
- TO PAUSE: touch .paused
- TO RESUME: rm .paused
- TO END: pkill -f tracker.py
